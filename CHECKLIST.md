# GitHub Repository-Checkliste (Delphi-Projekte)

## 1. Repository-Grundlagen (Pflicht)

[ ] Repository liegt in der richtigen **GitHub-Organisation**
[ ] Repository ist **privat** (sofern internes Projekt)

---

## 2. README.md (Pflicht)

[ ] README.md im Root des Repositories vorhanden und ausgefüllt
[ ] Projektbeschreibung (mind. 2–3 Sätze)
[ ] Delphi-Version angegeben

---

## 3. Projektstruktur (Pflicht)

[ ] Klare Trennung von:

* Quellcode
* Ressourcen
* Tools / Skripten
  [ ] Verzeichnisstruktur im README grob erläutert

---

## 4. Build & Ausführung (Pflicht)

[ ] Projekt lässt sich mit dokumentierter Delphi-Version öffnen
[ ] Abhängigkeiten sind dokumentiert (auch wenn sie alt sind)

---

## 5. Sicherheit & Compliance (Pflicht)

[ ] Keine Passwörter, Tokens oder Zertifikate im Repository
[ ] Konfigurationsdateien geprüft

---
